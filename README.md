**Q：电信、联通、移动用户该选哪个机房？**

A：电信用户优先 DC6 CN2 GIA-E；联通用户选 DC6 或荷兰 EUNL_9 均可；移动用户推荐 DC9 CN2 GIA。三网都有用户的情况下，CN2 GIA-E 套餐最保险，它同时接入三大运营商优质线路。

**Q：搬瓦工适合完全不懂 Linux 的人用吗？**

A：不太适合。搬瓦工是 self-managed VPS，没有可视化的 cPanel 面板，不提供操作指导。需要自己通过 SSH 管理服务器。如果有基本的 Linux 操作基础，照着网上教程操作完全没问题；如果完全零基础，建议先学一点 Linux 基础命令再入手。

**Q：CN2 GIA 遭受攻击会怎样？**

A：CN2 GIA 线路容量有限，遭受 DDoS 攻击时会对被攻击 IP 进行封堵（nullrouting），导致服务暂时不可访问。这是 CN2 GIA 线路的普遍特性，搬瓦工也不例外。攻击停止后通常会自动解封，或通过工单联系处理。

**Q：搬瓦工 KVM 20G 和 CN2 GIA-E 20G 配置差不多，为什么价格差这么多？**

A：两者配置参数接近，差距主要在**线路质量**和**可切换机房范围**。KVM 20G 走普通线路，晚高峰可能有波动；CN2 GIA-E 走三网优化专线，稳定性完全不是同一个级别。价格差距本质上是在为带宽成本付费。
